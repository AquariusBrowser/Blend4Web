euler_to_quat(euler, quat)

Convert euler rotation to quaternion rotation.

The euler angles have the following meaning (intrinsic system):

  • euler[0]: heading, y
  • euler[1]: attitude, z
  • euler[2]: bank, x

Using euler angles is discouraged, use quaternion instead.

Parameters:
Name Type Description
euler Float32Array Euler vector
quat Float32Array Destination quaternion vector
