功能¶

通用¶

  • 对任何复杂和尺寸的3D场景渲染效果

  • 数据格式设计，以适应WebGL规格和装载优化

  • 便捷的制作3D内容环境，通过Blender的插件来完成

纹理¶

  • 纹理贴图 如：应用一张图像到3D物体的表面

  • 多纹理 即：一个物体采用多重纹理

  • 渲染到纹理，RTT显示一个场景在另一个场景中和后处理效果

  • 各向异性过滤，AF是为提升在倾斜视角时看表面的质量，（要用到标准WebGL扩展）

  • 纹理压缩支持（S3TC / DXT格式）

  • 视频贴图 - 视频可以加载和播放，作为纹理层

  • 画布纹理 - 二维画布API可用于绘制的纹理

材质¶

  • 材质的透明性，如果需要的话通过深度排序（Z分级）

  • 增强立体表面细节的纹理（视差偏移映射方法）

  • 菲涅尔效应 - 反射对观看角度的依赖性

  • 动态反射

  • 支持材质节点

  • 光晕材质用来渲染光源和星星

照明¶

  • 多个光源

  • 光源类型 - 定向，半球，点，聚光

  • 漫射照明

  • 环境照明

  • 高光 - 从表面光反射

  • 环境贴图 - 从表面光反射

  • 法线贴图 - 附加表面细节的纹理

阴影¶

  • 静态阴影贴图（光照贴图）

  • 动态阴影映射

  • 自阴影 - 物体投射阴影在自己身上

  • 级联阴影贴图，CSM为大场景准备

  • 柔和阴影

粒子系统¶

  • 粒子系统实现效果，比如火，烟，飞溅等

  • 粒子系统的实例存储相似的物体：草，石头，树叶等

额外场景渲染¶

  • 雾

  • 天空和环境的天空盒

  • 镜头眩光效果

  • 水渲染

后处理效果¶

  • 运动模糊

  • 抗锯齿 - 图像边缘增强（快速近似抗锯齿，使用FXAA法）

  • 立体（视差图像法，需要3D眼镜）

  • 环境光遮蔽（SSAO方法时）

  • 景深，DOF

  • 曙暮辉（上帝射线）

  • 绽放 - 明亮的光线效果

  • 物体外轮廓

动画¶

  • 蒙皮 - 与骨骼系统物体变形

  • 动画物体，照相机和光源的位置，旋转和缩放

  • 骨骼动画（如角色的身体）

  • 顶点动画（如布料模拟）

  • 程序动画（如树叶风吹弯）

  • 纹理坐标动画（如可视化水波）

优化¶

  • 四椎体剔除 - 不可见的物体不渲染

  • 批处理，纹理地图 - WebGL的加载次数减少

  • 细节，LOD的水平 - 远的物体都那么详细

音频¶

  • 基于Web的音频API的音频引擎

  • 根据浏览器的各种文件格式的支持

  • 灵活的播放控制，声音暂停/恢复

  • 在三维空间中定位声源

  • 移动物体来关闭多普勒效应并用空间跳跃补偿

  • 灵活控制播放音量，速度和延迟

  • 淡入，淡出，静音

  • 高品质的声音循环

  • randomizing sound parameters to improve loop perception

  • 交叉推杆有声动画的支持

  • 动态压缩

  • 高效的长时间音轨存储和回放

  • 实时混音工具

物理¶

  • 刚体物理 - 碰撞检测，真实移动，重力，高度检测，扭转

  • 各种约束类型 - 刚性，铰链，弹簧，枢轴，滑动等

  • 光线追踪

  • 浮动和水下运动物理学

  • 轮式车辆模拟

  • 船舶模拟

事件驱动模型¶

  • 对于应用程序逻辑编写异步架构

  • 人物和动物的动画控制与人工智能

可视化编程¶

  • 逻辑编辑器工具允许使用基本模块组合逻辑链来创建交互式应用程序

其他¶

  • 数学曲线造型很长的物体（道路，电线，河）

  • 用鼠标在三维场景中拾取物体

  • 用于商业用途引擎的代码缩减和混淆处理

  • 模块结构的源代码

  • 强大的着色器预处理器的模块和功能模块（节点）支持

  • 方便的快速部署新的3D应用的系统

  • 范围广泛的设备支持选项

  • 用户手册和API文档

  • 用户交互 - 摄像头，角色，行为控制
